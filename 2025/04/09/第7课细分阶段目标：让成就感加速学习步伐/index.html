三、目标

  1. 托尔斯泰
    要有生活目标，一辈子的目标，一段时期的目标，一个阶段的目标，一年的目标，一个月的目标，一个星期的目标，一天的目标，一个小时的目标，一分钟的目标
  2. SMART原则的五项核心标准

​具体性（Specific）​
目标需清晰明确，避免笼统描述。例如，“提升团队效率”是模糊的，而“通过优化流程将团队月任务量提高20%”则符合具体性要求。具体目标需包含“做什么”“如何做”“涉及哪些资源”等细节。

​可衡量性（Measurable）​
目标需有量化指标或评估标准，便于跟踪进度。例如，“增加客户满意度”可改为“客户满意度调查得分从80分提升至90分”。数据化指标能直观反映目标的完成度。

​

可实现性（Achievable）​
目标应在资源、能力范围内可实现。例如，初创公司若设定“一年内市场份额超过行业龙头”可能不切实际，而“一年内市场份额提升5%”更符合可实现性。需评估时间、预算、人力等限制条件。

​相关性（Relevant）​
目标需与长期规划或整体战略一致。例如，销售团队的目标应服务于公司营收增长，而非孤立设定。若目标与其他任务冲突，可能导致资源浪费。

​时限性（Time-bound）​
目标需设定明确截止时间。例如，“完成产品升级”应改为“在2024年第三季度前完成产品升级”。时间限制能增强紧迫感，避免拖延。

四、林语堂

学习英语的唯一正轨，不出仿效与熟诵，仿效要是整句的仿效，热诵在效仿之后，必回环练习，最重要的，是学时必整句吞下，再整句吐出来，每日选二三句，回环熟诵。

注：一句一句学好–目标粉末化。

五、目标粉末化

  1. 让目标粉末化等于快速上瘾
  2. 每分钟都有成就感或者每分钟就有明显成就感，那么累积下来成就感源源不断地在刺激我们的大脑，那自然就上瘾了。

六、暴力突破

一天10句英语，21天210句（21篇）

  1. 早晨二三句，回环熟诵（上班路上）
  2. 中午二三句，回环熟诵（下班路上）
  3. 中午二三句，回环熟诵（上班路上）
  4. 晚上二三句，回环熟诵（下班路上）
  5. 晚上复习全天，回环熟诵，精准默写
  6. 复习以往全部，回环熟诵，错误之处，精准默写
  7. 继续推进，选四五句，回环熟诵

七、积累基础

越学越好学，越不学越难。

八、五篇（为一组）是一个里程碑

  1. 第一组：已经突破，建立自信，建立方法
  2. 第二组：挑战更大，训练元认知
  3. 第三组：进入细节，整合到自己的生命
  4. 第四组：浑然忘我，欲罢不能
  5. 第五组：整体回顾，构建强大的知识体系

九、《微习惯》

  1. 简单到不可能失败的自我管理法则
  2. 大幅缩减预期
  3. 成就感刺激行动
  4. 制定 → 执行 → 养成

十、上瘾

  1. 目标粉末化 = 频繁体验成功 = 上瘾
  2. 学到什么远远比学了什么更重要
  3. 学到什么说不清 = 慢性自杀
  4. 粉末，易消化吸收 → 超额完成
  5. 问自己学到了什么？

十一、思考题

  1. 你定的目标总是达成/达不成？原因是什么？
  2. 你有哪些成功达成目标的经历？
  3. 这个经历过程中哪些点（要点、亮点）是可以继承的？在之后的学习工作历程中是可以去复制的？

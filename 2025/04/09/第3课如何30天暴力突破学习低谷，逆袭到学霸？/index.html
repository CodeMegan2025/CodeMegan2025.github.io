第三课|如何30天暴力突破学习低谷，逆袭到学霸？

一、低谷

  1. Seth Godin
  2. 最初踏上学习之路，回报率很大。
  3. 不久，付出回报比骤降，进入低谷。
    原因：大部分都忘记了。
  4. 如果你能越过低谷，就能获得巨大回报。

二、低谷产生原因

  1. 看不清自己，未找到核心因果关系
  2. 是什么让我们陷入焦虑与绝望？收获不如起初那么大了。
  3. 为什么收获不如起初那么大？因为所学的东西大多数已经忘记了。
  4. 为什么所学的东西已经忘记了？因为没有及时复习。
  5. 为什么没有及时复习？因为要学的东西太多了。
  6. 那你打算怎么办？“……陷入思维混乱和挣扎，找不到办法。”

三、自我怀疑

失败 → 焦虑 → 挣扎 → 习得性无助 → 自我怀疑

四、Will·Smith

I think what happens is we make the situation more complex.

五、但问耕耘、莫问前程

莫问前路无知己，天下谁人不识君。

六、艾宾浩斯遗忘曲线

  1. 遗忘是残酷的。
  2. 只有高频回顾，就能极速推进。
  3. 遗忘时间表
    时间间隔 记忆量
    刚刚记忆完毕 100%
    20分钟之后 58.2%
  4. 20分钟之内复习
    回忆起来模糊，随时复习
  5. 只有高频回顾，就能极速推进
    有事看似多花时间，其实是在节省时间。
    元素周期表，原子质量。

七、神经科学

  1. 凡是你记住了任何东西：一定会对应产生相关的神经结构上面的变化，包括建立了新的神经回路链接和神经元的轴突、树突，长出了新的分支，甚至想出租车司机在大脑里面对应的这个城市的地图的脑区面积都比平常人要大。
  2. 学习不像长肌肉，学习就是长肌肉。
  3. 学习绝对不是处理抽象的知识货形象的图画。
  4. 必经途径就是重复，无论什么方式。
  5. 学习本质上是改造生命。

九、《刻意练习》

  1. 为什么有些人永远处在学习痛苦中？
  2. 舒适&学习&恐慌区

    舒适区：知识技能熟练掌握
    学习区：学习新知识、新技能
    恐慌区：暂时无法学会的知识技能

十、学习焦虑之源

  1. 只会做加减法却急于掌握量子学方程
  2. 中间有空缺 → 跳：挨摔
  3. 跳出舒适区：千百种悲剧之源
  4. 扩大舒适区：是学习唯一途径

十一、费曼学习法

  1. 你从头读，尽量往下读，直到你一窍不通时，再从头开始，这样坚持往下读，直至你完全读懂为止。
  2. 历久弥新
  3. 拉伸：学习区&恐慌区边缘
  4. 循序渐进往前推进

十二、杨振宁

  1. 只有两类数学书：你读不过第一句，以及你读不过第一页。
  2. 循序渐进，步步为营，是高效学习的唯一道路，能够实施的人获利；不能做到的人受到惩罚（习得性无助）。
  3. 弗里曼·戴森：我们懂得了学物理应该从平地开始，一块砖一块砖地砌，一层一层地加高，我们懂得了抽象化应在具体的基础工作之后，而绝非再它之前。

十三、少则得，多则惑

  1. 能积微者则速成。
  2. 我们学习效率低下，因为我们太聪明了，以为自己一口气能学很多东西；我们太傲慢了，以为学到一点东西是很容易的事情；我们太盲目了，无视世界上最伟大的头脑，以为自己比他们更厉害。所以一次次陷入思维混乱，学业挣扎，甚至人生危机。我们胆怯，不敢如实看待这个过程；我们畏惧，不敢直视真相，采取果断行动；我们贪婪，贪多嚼不烂，消化不良，所以厌食，还强迫自己继续吃。

十四、丢掉幻想、准备战斗

十五、张玲玲

一道题重复了11遍还犯错，继续做12遍，从第1遍到第12遍刷，不单增加了熟练度，而是把认知更加清晰化了，更加明朗化了，真正的差别，不是发生在考场上，也不是结果公布之后，而是在每天一点一滴学习过程中，很多人就已经被刷掉了，学习就是如此残酷，人生就是如此残酷。

十七、暴力突破学习

核心思维程序

  1. 第一章第一节满分（趋近100%）
  2. 第一章第二节满分（趋近100%）
  3. 综合复习满分（趋近100%）
  4. 以此类推
    注：循序渐进 → 快进 → 熟练 → 熟能生巧 → 上瘾 → 跃迁。

十八、思考题

  1. 你最近一次得“满分”是什么时候？
  2. 有没有哪个细小的环节（章节）是你能够得满分的？
  3. 还有哪些细小环节（知识点）是你能够得满分的？
  4. 如何高频的体验成功，激活上瘾回路？
